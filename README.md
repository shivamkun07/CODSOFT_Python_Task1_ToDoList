# 📝 Python To-Do List Application

## 📌 About
This project is developed as **Task 1** of the CodSoft Python Programming Internship.

The application helps users organize their daily tasks using a simple graphical interface built with Tkinter.

## ✨ Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- User-friendly GUI
- Lightweight and easy to use

## 🛠️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run
1. Clone this repository
2. Open the project folder
3. Run:
```bash
python todo.py
```
